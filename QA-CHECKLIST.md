# QA Checklist

- [ ] Login succeeds with approved Workspace account
- [ ] Courses load correctly
- [ ] CSV import creates correct number of sessions
- [ ] Validation errors are shown clearly
- [ ] Session edit screen saves changes
- [ ] Preview publish plan matches selected sessions
- [ ] Meet links are generated when required
- [ ] Classroom materials are created in correct course
- [ ] Scheduled publish triggers at correct time
- [ ] Duplicate posting is prevented
- [ ] Failed items are visible and retryable
- [ ] Posting log is readable and complete
